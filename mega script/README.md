How to start PoochBot
1. Download source from github: https://github.com/TheRealMrBones/PoochBot (store poochbot in C drive!)
2. Run the file PoochBot/PoochBotNinite.exe to download all dependencies (If some fail check next section)
3. Open VS Code or Command Prompt as administrator and open the PoochBot folder
4. IF USING VS CODE: Go to Terminal > New Terminal
5. Enter "pip install -r requirements.txt"
6. Enter "py main.py" to start PoochBot!
